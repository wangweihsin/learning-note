## 👻week4
### Insertion Sort
- 插入排序法
- 重複把已排序的元素往後放
-時間複雜度
  - 最佳時間複雜度：O(n)
  - 平均時間複雜度：O(n^2)
  - 最差時間複雜度：O(n^2)
  - 空間複雜度：O(1)
  - Stable sort：是
 - [補充連結](https://emn178.pixnet.net/blog/post/93791164)
 - [PPT](https://docs.google.com/presentation/d/e/2PACX-1vQOTMDM-5-OUaGfnLUOFVgefFwSVRplSwnbicp0CXOQrB5H8RM_1Aq8o_4JxHlncEmhjvqk3tzcoB7s/pub?start=false&loop=false&delayms=3000&slide=id.g619c41adaf_0_22)
 
## 👻week5
### Quick Sort
- 快速排序法
- 從數列中抽一個數字當標準(pivot)，比標準大放右邊，比標準小放左邊，這樣pivot就會落在他應該在的位置。接著把左右兩堆看成新的數列，再從中抽數字當表準，重複直到分不出兩堆，所有數字都落在他該在的位置，排序就完成了。
- [我的筆記](https://github.com/wangweihsin/learning-note/tree/master/01.QuickSort)
- [PPT](https://docs.google.com/presentation/d/e/2PACX-1vSqz8sTxT4xyjgiz-htLvZd7FZ_5ZzgKf60pFEoNLU5S77JxrsGJ2vd15CdxlfLtT3g2aizHP-Ebk9b/pub?start=false&loop=false&delayms=3000&slide=id.p)
 
## 👻week6
### Heap Sort
- 堆積排序法
- 像樹狀圖一樣，每個父母下面有兩個孩子，最底層可以有缺(只有一個或沒有)。父母必須比子女大(小)，最上層的父母最大(小)。
- [我的筆記](https://github.com/wangweihsin/learning-note/tree/master/03.HeapSort#heap-sort-%E5%A0%86%E7%A9%8D%E6%8E%92%E5%BA%8F%E6%B3%95)
- [PPT](https://docs.google.com/presentation/d/e/2PACX-1vRAGwnUvg6BcXoML5u9f4gO6YKcz0vXf7bDnPho_S7mG5D0SBR78djt91RKUPMxqNfkVIcu3l5WCXPh/pub?start=false&loop=false&delayms=3000&slide=id.p)

## 👻week7
### Merge Sort
- 合併排序法
- 將大的問題分解成很多個小問題，從小的問題開始解決起。把小事解決完以後合併回大問題，大問題自然就解決了。
- [我的筆記](https://github.com/wangweihsin/learning-note/tree/master/02.MergeSort)
- [PPT](https://docs.google.com/presentation/d/e/2PACX-1vToxkEzc1H1RT5MI9G941KQFBC7GO_Efn95wTqXLEdr3LDBSNcQb-M46IOC-_RzZih6IBEwwy3rWQuE/pub?start=false&loop=false&delayms=3000&slide=id.p)
